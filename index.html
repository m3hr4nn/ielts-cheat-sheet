Question Types (14 types)

1. Multiple Choice

  • Strategy: Eliminate wrong options, watch for paraphrasing
  • Trap: Distractor options use text words but wrong meaning

2. True/False/Not Given ‚≠ê MOST DIFFICULT

Answer When to Choose Example
TRUE Statement matches text (even if paraphrased) Text: "Many scientists support this" / Statement: "Numerous researchers agree" = TRUE
FALSE Statement contradicts text directly Text: "All birds fly" / Statement: "Some birds cannot fly" = FALSE
NOT GIVEN Info not mentioned OR need assumptions Text: "Birds migrate" / Statement: "Birds migrate in winter" = NG (season not specified)

Golden Rule: If you're thinking "Maybe..." or "Probably..." = NOT GIVEN!

3. Yes/No/Not Given

Same as T/F/NG but for opinions/claims/views. Text must explicitly state the opinion.

4. Matching Headings

  • Read all headings first
  • Focus on MAIN idea of paragraph (first 2 + last sentences)
  • Eliminate used headings
  • Ignore distractor headings (minor details mentioned)

  • Paragraphs can be used MORE THAN ONCE or NOT AT ALL
  • Questions NOT in order
  • Scan for keywords and synonyms

  • Matching Features - names, dates, theories (scan for specific names/dates)
  • Matching Sentence Endings - grammar must fit correctly
  • Sentence completion
  • Summary completion (from text or word bank)
  • Note/table/diagram completion
  • Short answer questions

True/False/Not Given - Detailed Guide

This is the MOST CHALLENGING question type. Here's how to master it:

When to Choose TRUE:

  • Statement matches text meaning (even with different words)
  • Statement is a logical conclusion from text facts
  • Paraphrasing is accurate

Example:

Text: "The majority of teenagers use smartphones daily."

Statement: "Most young people use mobile phones every day." = TRUE

When to Choose FALSE:

  • Statement directly contradicts text
  • Opposite information given
  • Different numbers/dates/facts

Example:

Text: "The study involved 500 participants."

Statement: "The research included 5,000 people." = FALSE

When to Choose NOT GIVEN:

  • Information not mentioned at all
  • Need to make assumptions
  • Specific details not provided
  • Related topic mentioned but specific claim not stated

Example:

Text: "Pollution levels are increasing in major cities."

Statement: "Pollution levels are increasing due to car emissions." = NOT GIVEN (cause not mentioned)

Writing Task 2

40 minutes250+ words

Structure (4 Paragraphs)

  1. Introduction - Paraphrase + Thesis statement (40-50 words)
  2. Body 1 - Main argument with P.E.E.L. (80-100 words)
  3. Body 2 - Second argument with P.E.E.L. (80-100 words)
  4. Conclusion - Summarize + restate position (40-50 words)

Four Essay Types

1. Opinion (Agree/Disagree)

  • Clear position in introduction
  • 2 body paragraphs supporting your view
  • Restate position in conclusion

2. Discussion (Both Views)

  • ¬∂2: First viewpoint
  • ¬∂3: Second viewpoint + your opinion
  • Show which side you support

3. Advantages/Disadvantages

  • ¬∂2: Advantages (2-3 points)
  • ¬∂3: Disadvantages (2-3 points)
  • State which outweighs (if asked)

4. Problem-Solution

  • ¬∂2: Problems (2 problems with explanations)
  • ¬∂3: Solutions (2 solutions with expected outcomes)

P.E.E.L. Body Paragraph Structure

  • Point - Topic sentence
  • Explain - Develop the idea
  • Example - Specific evidence
  • Link - Connect back to thesis

Grammar Terms Explained

Base Verb (Infinitive without "to")

The simplest form of a verb, without any tense or conjugation.

Examples: work, study, go, eat, write

Past Participle

The verb form used with perfect tenses and passive voice. For regular verbs, add -ed. Irregular verbs have unique forms.

Regular: worked, studied, played

Irregular: gone, written, taken, eaten, been

Present Participle

The verb form ending in -ing, used in continuous tenses.

Examples: working, studying, going, eating, writing

Gerund

A verb ending in -ing that functions as a noun. While identical in form to the present participle, gerunds name activities or states.

Examples: "Swimming is healthy." / "I enjoy reading."

Note: In "I am swimming," swimming is a present participle. In "Swimming is fun," swimming is a gerund.

Auxiliary Verbs (Helping Verbs)

Verbs that help form tenses, questions, and negatives.

Common auxiliaries: am, is, are, was, were, have, has, had, will, would, shall
